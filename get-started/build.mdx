## Step 1: Create Project

1. After signing up, you will land on the Projects page in FlowScale, and that will look like this, [Blank image of flowscale repos]
2. Create a new Project, by clicking on “Create ComfyUI Project” button. And, once you do that a dialog box will pop up asking you for relevant information to create a project.
*Note: This project is a actually a GitHub repository that you can collaborate with your peers or teams with.* 
[Image of Comfy Create project]

## Step 2: Start ComfyUI Editor

1. Once you have created and entered a project, you will see the “Editor” page. [Screenshot of the Editor]
2. To start ComfyUI, click on “Start ComfyUI” button [Image showing start ComfyUI]
3. To see the logs, click on the logs bar at the bottom of the page and once it’s expanded, you can see all the System logs showcasing logs of ComfyUI and the machine it is running on.
4. Once your ComfyUI has started you will see a screen like this, [Image showing editor page with comfyui started]

## Step 3: Create a new workflow

1. Once ComfyUI is started, create a new workflow by clicking on `+` button > “Create New Workflow” option, in the Workflows Sidebar. [Show the dropdown]
2. Once you click on that button, it will load a new unsaved workflow in the ComfyUI Editor. [Show untitled workflow]

## Troubleshooting

Here's how to solve some common problems when working with the CLI.

<AccordionGroup>
  <Accordion title="Mintlify is not loading">
    Update to Node v18. Run `mintlify install` and try again.
  </Accordion>
  <Accordion title="No such file or directory on Windows">
Go to the `C:/Users/Username/.mintlify/` directory and remove the `mint`
folder. Then Open the Git Bash in this location and run `git clone
https://github.com/mintlify/mint.git`.

Repeat step 3.

  </Accordion>
  <Accordion title="Getting an unknown error">
    Try navigating to the root of your device and delete the ~/.mintlify folder.
    Then run `mintlify dev` again.
  </Accordion>
</AccordionGroup>
