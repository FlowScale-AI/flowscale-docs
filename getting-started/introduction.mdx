## What is FlowScale AI?

FlowScale AI bridges the gap between AI experimentation and production deployment. Whether you're an AI artist, developer, or founder, FlowScale AI provides:

- **Cloud-native ComfyUI**: Run workflows on powerful GPUs (T4, L4, A100, H100)
- **Instant API deployment**: Convert workflows to REST APIs in seconds
- **Team collaboration**: Centralized workspace with version control
- **Interactive UIs**: Generate Gradio interfaces automatically

<Steps>
  <Step title="Sign up">
    Create your FlowScale AI account at [app.flowscale.ai](https://app.flowscale.ai)
  </Step>
