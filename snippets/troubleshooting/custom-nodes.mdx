<Accordion title="Custom Node Not Found">
**Solution**: Ensure the node was properly detected during import. Check the custom nodes section in your project settings. If missing, try re-importing the workflow or manually installing the custom node.
</Accordion>

<Accordion title="Node Installation Failed">
**Solution**: Check the System Logs during ComfyUI startup for installation errors. Some custom nodes may have dependency conflicts. Contact support with the specific error message if issues persist.
</Accordion>

<Accordion title="Node Version Conflicts">
**Solution**: If conflicting nodes are detected during import, select the correct versions that your workflow actually uses. You can also check the custom node's GitHub repository for compatibility information.
</Accordion>
