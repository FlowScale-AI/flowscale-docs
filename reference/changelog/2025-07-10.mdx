### Critical Fixes

- **Fixed**: Memory leaks in long-running workflows causing pod crashes
- **Fixed**: Inconsistent model loading behavior across different GPU types
- **Fixed**: API rate limiting edge cases causing legitimate requests to be blocked
- **Fixed**: Workflow export/import issues with custom node dependencies

- **Fixed**: Property panel not updating when switching between nodes quickly

- **Fixed**: API documentation not reflecting latest parameter changes
- **Fixed**: CORS issues when calling APIs from certain domain configurations
